# TriRC-Net

Point cloud registration aims to recover the rigid transformation between partially overlapping 3D scans and is a fundamental component in 3D vision, robotics, and scene reconstruction. Although recent coarse-to-fine learning frameworks have achieved promising performance, they still suffer from insufficient patch recall, ambiguous local assignment, and unreliable geometric verification under low-overlap conditions. Thus, for low-overlap point cloud registration, we propose TriRC-Net, a recall-and-consistency-based network framework. We design a recall-preserving patch proposal generator (RPPG) in the coarse matching sub-network to retain potential true patch correspondences and suppress isolated false proposals; simultaneously, a Sinkhorn local optimal matching (SLOM) module is developed in the local assignment sub-network to enable rejectable point matching and alleviate one-to-many and many-to-one ambiguities. Moreover, to address residual outlier amplification in pose estimation, we develop a triadic graph-consistency dual-stage geometric refinement (TGC-DGR) module, which constructs spatially compatible triadic hypotheses for robust initialization and further performs residual refinement by fusing sparse high-confidence and fine local correspondences.
